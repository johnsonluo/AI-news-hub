# AI News Hub

A daily AI news aggregator and summary generator built with Next.js, Tailwind CSS, and TypeScript.

## Features

- **News Feed**: Browse the latest AI news by category (LLM, Computer Vision, Industry, etc.).
- **Daily Summary**: Get a concise summary of the day's top AI stories.
- **Shareable**: One-click download of the daily summary as an image or copy text for social media sharing.

## Getting Started

1.  **Install dependencies**:

    ```bash
    npm install
    ```

2.  **Run the development server**:

    ```bash
    npm run dev
    ```

3.  **Open in browser**:
    Navigate to [http://localhost:3000](http://localhost:3000) to see the application.

## Project Structure

- `src/app`: Next.js App Router pages and layouts.
- `src/components`: Reusable UI components.
- `src/lib`: Types, utility functions, and mock data.

## Technologies Used

- Next.js 14
- React 18
- Tailwind CSS
- Lucide React (Icons)
- html2canvas (Image generation)
- Day.js (Date formatting)
